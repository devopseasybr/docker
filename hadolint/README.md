# Hadolint
Ferramenta linter para garantir a correção da sintaxe do Dockerfile, adesão às melhores práticas e a identificação de possíveis problemas relacionados à criação de imagens Docker.

## Sobre esse projeto
Esse projeto cria uma imagem Docker customizada com a ferramenta Hadolint. 
A mesma é utilizada no Stage build da pipeline CI/CD para verificar a sintaxe do arquivo Dockerfile.
